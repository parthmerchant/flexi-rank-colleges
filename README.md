# Flexi Rank Colleges
-------------------------------------
Information Design and UX Project for filtering colleges based on acceptance rate, SAT score as well as other attributes using Pandas, Matplotlib, NumPy and Tkinter

The dataset is quite large and can be downloaded from: https://data.ed.gov/dataset/college-scorecard-all-data-files-through-6-2020/resources

Click on the video demo to see how the project works!
